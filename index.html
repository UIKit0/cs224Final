CS224 Final

Interactive Non-Photorealistic Rendering

Download this project as a .zip file

Description:

For our final project we implemented a non-photorealistic shader based on Gooch et al. [1999] with silhouette edge detection using geometry shaders. As well, we implemented a non-photorealistic based smoke effect and its physical simulation using ODE (physics engine) according to McGuire et al. [2006] Our final product was a simple airplane shooter game for showcasing the effects and shaders we created.
